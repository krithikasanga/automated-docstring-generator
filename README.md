# Automated Python Docstring Generator

A FastAPI + Streamlit project that:
- Uploads Python files
- Parses structure using AST
- Extracts functions and class methods
- Prepares for AI-based docstring generation

## Tech Stack
- FastAPI
- Streamlit
- Python AST

## How to Run

### Backend
uvicorn main:app --reload

### Frontend
streamlit run app.py
